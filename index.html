Launch a Module (LM) - eYRC 2016


Launch a Module (LM) is an eYRC Theme from eYRC-2016. The theme problem statement required the students to program a robot to recognize objects based on their shape, size, and color using an over-head camera; then autonomously deposit the objects on their corresponding marker in the arena while avoiding obstacles. The programming languages used for this theme were Python and Embedded C


Challenges

  1. 1. Recognising the different shapes, sizes and colors.
  2. 2. Navigating through the arena avoiding obstacles using overhead camera feed.
  3. 3. Finding the shortest path to the destination.
  4. 4. Picking and Placing objects in the arena based on shape and color.


My Role in the theme:
As one of the theme developers, I was responsible for creating the initial tasks that were given to the students who participated in the competition. The initial tasks were based on learning basic python programming and image processing techniques like shape and color detection using OpenCV library in python. In addition to this I also provided support to the participating students to help them complete the given tasks through the online piazza forum and live sessions during the competition.
